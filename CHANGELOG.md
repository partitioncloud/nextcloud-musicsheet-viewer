## [1.3.0] - 2026-04-19

### Fixed

- MuseScore >v4.3 support https://github.com/partitioncloud/nextcloud-musicsheet-viewer/issues/3 https://github.com/CarlGao4/webmscore/pull/1

## [1.2.5] - 2026-03-08

### Fixed

- Further reduce app size https://github.com/partitioncloud/nextcloud-musicsheet-viewer/issues/1

## [1.2.4] - 2026-03-06

### Fixed

- App won't run because of CSPs https://github.com/partitioncloud/nextcloud-musicsheet-viewer/issues/2
- Reduce app size https://github.com/partitioncloud/nextcloud-musicsheet-viewer/issues/1

## [1.2.2] - 2026-03-02

### Added

- `.mscx` support

## [1.2.1] - 2026-02-28

### Added

- Mimetypes icons (heavily inspired by [files_3dmodelviewer](https://github.com/WARP-LAB/files_3dmodelviewer))

## [1.2.0] - 2026-02-26

### Fixed

- Public shares

## [1.1.0] - 2026-02-24

### Added

- Ability to load any file [potentially supported by MuseScore](https://musescore.org/en/handbook/3/file-formats#share-with-other-software)
